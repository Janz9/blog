Selenium 入门

本文字数： 2.7k 阅读时长 ≈ 2 分钟

入门

Selenium 是一个 Web 自动化测试工具, 支持多语言, 多平台, 多浏览器

组件

Selenium WebDriver 支持市场上所有主流浏览器, 尽可能模拟真实用户与浏览器的交互, 在不同的浏览器中, 这可能在不同的级别上发生, 下载地址 webdriver

Selenium Grid 允许测试将命令路由到远程浏览器实例, 旨在提供一种在多台计算机上并行运行测试的简便方法, 并集中管理不同的浏览器版本和浏览器配置, 下载地址 grid

  • hub 集线器中枢, 具有已注册服务器的列表, 可以访问并允许控制这些实例
  • node 节点中枢, 执行测试的实例, 作为节点的机器不必与集线器或其他节点具有相同的平台或浏览器选择

Selenium IDE 用于开发 Selenium 测试用例的工具, 易于使用的 Chrome 或 Firefox 扩展程序, 通常是开发测试用例的最有效方法, IDE 使用现有的 Selenium 命令以及由该元素的上下文定义的参数, 在浏览器中记录用户的操作并支持导出多种语言, 下载地址 ide

Mock 浏览器

HtmlUnit

HtmlUnit 是用于 Java 程序的无 GUI 浏览器, 它为 HTML 文档建模, 并提供允许调用页面, 填写表单, 单击连接等API, 它具有 JavaScript 支持, 并且能够与 AJAX 库一起使用, 根据所使用的配置来模拟 Chrome, Firefox 或 IE

官方文档 HtmlUnit

PhantomJS

PhantomJS 是基于 Webkit 的无头浏览器, 从历史上看, 这是一个流行的选择, 但 PlantomJS 项目已不再维护, 现在 Chrome 和 Firefox 已提供无头运行的功能
